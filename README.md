# Number Guessing Game (Python)

This is a **simple console-based Number Guessing Game** built using **Python**.  
In this game, the computer generates a random number, and the user tries to guess it.

The program gives hints like **Too High** or **Too Low** until the correct number is guessed.

## ðŸ”¹ How the Game Works
1. The computer selects a random number between **1 and 100**
2. The user enters a guess
3. The program checks the guess:
   - Too high
   - Too low
   - Correct
4. The game ends when the user guesses the correct number
